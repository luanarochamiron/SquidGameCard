# Squid Game Card with 3D Flipping Animation

Um componente interativo de cartão temático inspirado na série Round 6 (Squid Game), desenvolvido exclusivamente com HTML5 e CSS3. O projeto destaca o uso de propriedades avançadas de animação em CSS para criar um efeito de rotação tridimensional (3D Flip) realista quando o usuário passa o mouse sobre o elemento.

## Tecnologias Utilizadas

* HTML5: Estruturação dos contêineres para as duas faces do cartão (frente e verso).
* CSS3: Estilização completa, paleta de cores temática e definição de formas geométricas.
* Efeitos 3D em CSS: Manipulação de perspectiva tridimensional através de propriedades como `perspective`, `transform-style: preserve-3d` e `backface-visibility`.

## Funcionalidades e Técnicas Aplicadas

* Animação Flip 3D: Rotação suave de 180 graus no eixo Y ao passar o mouse (hover effect), revelando o verso do cartão com um comportamento tridimensional imersivo.
* Formas Geométricas Simbólicas: Renderização precisa dos icônicos símbolos da série (Círculo, Triângulo e Quadrado) centralizados na superfície do cartão.
* Minimalismo e Fidelidade Visual: Design limpo com cantos retos e bordas nítidas, reproduzindo a estética misteriosa do cartão de convite original.
* Centralização Perfeita: Uso de CSS Flexbox para alinhar horizontal e verticalmente o cartão na tela sobre um plano de fundo harmonioso.
